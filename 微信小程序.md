## 组件中无法加载到全局样式
在组件的js中添加以下代码
```
options: {
    addGlobalClass: true
}
```
## bindtap传参
```
## wxml
<list wx:for="{{list}}" wx:for-item="item"  data-gid="1"  bindtap="detail" ></list>
## js
detail: function(e) {
    console.log(e.currentTarget.dataset.gid)
}
```
